# Food Stand Texture Overrides

Generated in-repo for Kingdom Sim's Food Stand texture polish pass.  These CC0 textures use online low-poly fantasy market references for style direction only; no third-party image pixels are copied.

## Files

| File | Purpose | Applied to |
|------|---------|------------|
| `food_stand_wood_stall.png` | Warm reclaimed-wood planks | `stall-fantasy-town.glb`, `stall-green-fantasy-town.glb`, `stall-stool-fantasy-town.glb` |
| `food_stand_canvas_awning.png` | Cream/tan canvas stripes | `banner-green-fantasy-town.glb` |
