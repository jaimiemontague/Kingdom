Kingdom Sim UI Pack (CC0)

This pack provides a cohesive, pixel-crisp UI skin for WK7.

Folders:
- panels/ (9-slice textures)
- buttons/ (button states)
- icons/ (small UI icons)

9-slice contract:
- Panel textures are 32x32
- Slice border is 4px (top/bottom/left/right)
- Scale with nearest-neighbor (no smoothing)

Button contract:
- Button textures are 24x24
- Use normal/hover/pressed
- Scale with nearest-neighbor

Icon contract:
- Icons are 16x16
- Use nearest-neighbor

Palette notes:
- Dark frame with warm mid tones (Majesty-inspired)
- Use ink outlines for readability on terrain
