# Hero Sprites

Engine-loaded folder structure:

`assets/sprites/heroes/<hero_class>/<action>/*.png`

WK2 Build B target:
- Classes: `warrior`, `ranger`, `rogue`, `wizard`
- Actions: `idle`, `walk`, `attack`, `hurt`, `inside`

Naming: `frame_000.png`, `frame_001.png`, ...

See: `docs/art/hero_sprite_pipeline.md`
